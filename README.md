## Tipos de variables
1) let - Variable reasignable - Forma mas comun de crear variables
2) const - Variable NO reasignable - Ideal para valores que NO deben cambiar

A la variable se le debe colocar un nombre y asignar un valor

## Tipos de Datos
### Datos Primitivos
1) string - Se usa para representar una cadena o texto - Se representa con comilla ('), doble comillas (") o acento grave (`)
```js
'cubo'
"casa"
`programador` 
```
2) number - Se usa para representar un numero - Se escribe en numero regular
```js
1
-10
0.85
```
3) boolean - Valor logico que solo puede ser true o false
```js
true false
```
4) null - Valor vacio - Se representa con el texto null sin comillas
```js
null
```
5) undefined - Variable declarada pero sin valor - Se representa con el text undefined sin comillas
```js
undefined
```

### Datos NO primitivos
1) object - Coleccion clave-valor o key-value - Se representa con llaves ({ }) - Cada key-value es separado por una coma (,)
```js
{
  nombre: 'alejandro',
  direccion: '123 Walhalla',
  telefono: '123-123-1234'
}
```
2) array - Lista ordenadas - Se representa con corchertes ([ ]) - Una lista puede ser definida como una lista de textos, numeros o cualquier otro tipo de variable incluso una lista de listas
```js
[
  'alejandro',
  'carlos',
  'erik'
]
```
